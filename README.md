
# Fazer_ Portfolio

Hi there, I'm Luke and this is my first and most simple web portfolio. This site was made with pure html/css/js. 

## Some small features I've added :P

 - Pong that plays itself
 - Nice css animations
 - Proper responsiveness
 - w a v e s

## Why

This portfolio was created because it's started becoming a little bit annoying to keep track of projects, even though many of mine are unfinished. This website helps organize my projects and personal info into nice, readable information. The design was inspired by a couple of portfolios I found that I wanted to simplify. I originally wanted to learn three.js in order to make my portfolio, but it proved to be too time consuming to learn so I went with the tried and true basic method. I wanted the site to have depth to it so I added animations every now and then, and they spice things up a bit. I've used heikei before to play with waves, so it's been really fun to incorporate them into an actual site and have it look okay:
